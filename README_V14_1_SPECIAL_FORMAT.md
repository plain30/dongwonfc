# 동원FC 회비관리시스템 V14.1 — 특별회비 수정 양식 지원

## 지원 양식
`templates/특별회비내역_양식수정.xlsx`와 동일한 구조를 그대로 업로드할 수 있습니다.

- 1행: 경조사일자
- 2행: 경조사내용
- 3행: 성명 / 회원번호 / 각 경조사 납부금액 / 남은금액 / 특별회비 합계
- 납부: 숫자 금액(예: 10000)
- 미납: `-` 또는 빈칸
- 본인 경조사: `본인경조사`

## 대시보드 표시
특별회비 현황 표도 업로드 파일과 같은 3단 헤더 구조로 표시합니다.

1. 경조사일자
2. 경조사내용
3. 회원별 납부금액, 남은금액, 특별회비 합계

회원 성명과 회원번호 열은 좌측에 고정되어 가로 스크롤 중에도 확인할 수 있습니다.

## 적용
이 버전은 프런트엔드의 엑셀 인식 및 화면 배치를 변경합니다. Apps Script는 기존 V13.1/V14 API와 호환되지만, 폴더 전체를 GitHub에 덮어쓰는 것을 권장합니다.
