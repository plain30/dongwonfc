# 동원FC 2026 회비납부현황 대시보드

## 실행 방법
1. 압축을 풉니다.
2. `index.html`을 더블클릭해서 실행합니다.
3. GitHub Pages에 올릴 때는 이 폴더 전체를 업로드하면 됩니다.

## 로그인
- 관리자: `admin` / `admin2026`
- 회원: `본인이름` / `휴대전화 뒤 4자리`

## 포함 기능
- 관리자 전체 조회
- 회원 본인 정보만 조회
- 회원 추가/삭제/수정
- 회원별 관리자 권한 부여/해제
- 비밀번호/휴대폰/회비/특별회비 수정
- CSV 다운로드
- 다크모드
- 수정 데이터 localStorage 저장

## 주의
이 파일은 서버 없는 정적 HTML입니다. 권한과 비밀번호는 브라우저 저장소에 저장되는 편의 기능이며, 실제 보안이 필요한 경우 서버/DB 기반 로그인으로 전환해야 합니다.

생성 데이터: 회원 72명, 특별회비 행사 7건
