# 동원FC 회비관리 시스템 v8

## 적용 내용
- 폴더 구조 분리: css / js / assets
- 회원번호 000 및 공백 번호 자동 제외
- 관리자: admin / admin2026
- 한재식 이름 로그인 차단
- 회원: 본인이름 / 휴대전화번호 뒤 4자리
- 회원정보, 회비내역, 특별회비 엑셀 업로드 반영
- 관리자 대시보드, 회원 내현황 원형그래프
- JSON 백업/복원, CSV 다운로드

## GitHub Pages 업로드
압축 해제 후 index.html, css, js, assets 폴더를 모두 dongwonfc 저장소에 업로드하세요.
기존 데이터가 남아 있으면 관리자 로그인 후 "파일 업로드 > 기본값 초기화"를 1회 실행하세요.
