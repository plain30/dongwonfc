# 동원FC 회비관리 시스템 v9.1

## 회비 계산 기준
- 회원회비: 월 20,000원 × 12개월 = 연간 내야할금액 240,000원
- 특별회비: 경조사 1건당 10,000원
- 본인경조사는 해당 회원의 내야할금액에서 제외
- 관리자 환경설정에서 금액을 변경하면 전체 회원 금액과 그래프가 자동 재계산됨

## 주요 변경사항
- 회비현황: 납부금액 오른쪽에 `내야할금액`, 그 오른쪽에 `남은금액` 표시
- 특별회비현황: 납부금액 오른쪽에 `내야할금액`, 그 오른쪽에 `남은금액` 표시
- 관리자 및 회원 원형그래프에 납부금액/내야할금액/남은금액 표시
- 회원 내 상세금액에 총 내야할금액 표시
- 관리자 환경설정 메뉴 추가
- 회원번호 000 자동 제외 및 권한 분리 유지
- 저장소 키를 v9.1로 변경하여 이전 버전 캐시 충돌 방지
