# 동원FC 회비관리시스템 V11.2 — Google Sheets 정식판

Firebase/Google Cloud 유료 결제 없이 Google 스프레드시트와 Apps Script를 공동 데이터 저장소로 사용합니다.

## 이미 반영된 설정
- Apps Script 웹 앱 주소가 `js/config.js`에 입력되어 있습니다.
- 관리자 로그인 ID: `admin`
- 권장 관리자 비밀번호: `admin2026`

## 최초 1회 작업
1. Google 스프레드시트에 연결된 Apps Script 편집기에서 이 폴더의 `apps-script/Code.gs` 전체 내용을 붙여넣고 저장합니다.
2. Apps Script에서 **배포 → 배포 관리 → 수정(연필)** 후 새 버전으로 배포합니다.
3. 실행 사용자: **나**, 액세스 권한: **모든 사용자**로 설정합니다.
4. GitHub 저장소에 프로젝트의 모든 파일과 폴더를 업로드합니다.
5. `https://plain30.github.io/dongwonfc/setup.html?v=111`에 접속합니다.
6. **서버 연결 확인**을 누른 뒤 `admin2026`으로 초기설정을 실행합니다.
7. 로그인 화면에서 `admin / admin2026`으로 접속합니다.
8. 관리자 화면에서 회원정보·회비내역·특별회비 엑셀을 업로드합니다.

## 중요: Code.gs를 수정한 뒤
Apps Script는 저장만 해서는 기존 웹 앱에 반영되지 않을 수 있습니다. 반드시 **배포 관리에서 새 버전으로 다시 배포**해야 합니다. 기존 `/exec` 주소를 유지하려면 새 배포를 만들지 말고 기존 배포를 수정하세요.

## 기능
- 여러 PC/휴대전화 공동 데이터
- 관리자 전체 조회, 회원 본인 자료만 조회
- 회원정보·회비·특별회비 엑셀 업로드
- 월 회비 20,000원 × 12개월
- 특별회비 경조사 1건당 10,000원
- `면제`, `EXEMPT` 회비면제 처리
- 회원번호 `000`, 탈퇴회원 제외
- 변경 이력 기록
- 15초 자동 동기화

## 파일 구성
- `index.html`: 로그인
- `admin.html`: 관리자 화면
- `member.html`: 회원 화면
- `setup.html`: 최초 초기설정 및 서버 연결 확인
- `apps-script/Code.gs`: Google Apps Script 서버 코드
- `js/config.js`: Apps Script `/exec` 연결 주소
