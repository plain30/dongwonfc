# 동원FC 회비관리시스템 V14

V14는 현재 사용하는 `2025-2026년 동원FC 특별회비현황` 엑셀 양식을 그대로 업로드할 수 있도록 특별회비 처리 구조를 개편한 버전입니다.

## 특별회비 파일 자동 인식
- `경조사 일자` 행
- `경조사 내용` 행
- 건별 금액 행
- 회원번호 / 직책 / 성명
- `납부`, `-`, `본인경조사`, `면제` 상태
- 비고의 `정지회원`은 특별회비 반영에서 제외

업로드 후 경조사 날짜·내용·건별금액·회원별 상태가 Google Sheets에 저장되고 관리자/회원 화면에 공동 반영됩니다.

## 적용
1. `apps-script/Code.gs`를 Apps Script에 덮어씁니다.
2. 배포 관리에서 **새 버전**으로 재배포합니다. 실행 사용자 `나`, 액세스 권한 `모든 사용자`.
3. GitHub 저장소에 이 폴더의 모든 파일을 덮어씁니다.
4. `https://plain30.github.io/dongwonfc/admin.html?v=141` 접속 후 Ctrl+F5.
5. 파일 업로드 → 특별회비 업로드에서 현재 엑셀 파일을 선택합니다.

> V14부터 특별회비현황 시트의 열은 회원번호, 경조사, 경조사일자, 부과금액, 상태로 확장됩니다. 따라서 Apps Script Code.gs 재배포가 필수입니다.
