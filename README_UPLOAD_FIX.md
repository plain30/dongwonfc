# V14 파일 업로드 동기화 수정

## 핵심 변경
- JSONP 한 요청 크기를 4,200자에서 1,200자로 축소
- 실패한 조각은 최대 3회 자동 재시도
- 업로드 진행률 표시
- Apps Script CacheService 대신 숨김 시트 `_UPLOAD_CACHE`에 조각을 안전하게 저장
- 저장 직후 서버 자료를 다시 읽어 회원 수를 검증
- 검증이 실패하면 성공 메시지를 표시하지 않고 구체적인 오류를 안내

## 반드시 함께 적용
1. `apps-script/Code.gs` 전체 교체
2. Apps Script 배포 관리에서 **새 버전**으로 재배포
3. GitHub 저장소에 전체 파일 덮어쓰기
4. `admin.html?v=131`로 접속 후 Ctrl+F5
