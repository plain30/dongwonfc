# V14 특별회비 업로드 안내

현재 파일 `2026년특별회비내역.xlsx`의 첫 번째 시트 형식을 그대로 지원합니다.

- 전년도 미수금도 하나의 경조사 항목으로 보존합니다.
- `납부`는 납부금액에 포함합니다.
- `-` 또는 빈칸은 미납으로 처리합니다.
- `본인경조사`와 `면제`는 내야할금액에서 제외합니다.
- 각 경조사의 금액은 파일의 금액 행을 우선 사용하고, 없으면 환경설정의 건당 금액을 사용합니다.
- 회원번호를 우선 매칭하고, 번호가 일치하지 않으면 성명으로 한 번 더 매칭합니다.

## 검증된 업로드 구조
업로드해주신 파일에서 다음 7개 항목이 인식됩니다.
- 전년도 미수금 1건: 장남성 회원 모친상
- 2026년도 특별회비 6건: 김기철 회원 모친상, 김대훈 회원 결혼, 박상언 회원 부친상, 정민안 상위 빙모상, 이강인 고문 자녀결혼, 윤삼봉 회원 자녀결혼

전년도 미수금 열의 `-`는 새 미납으로 부과하지 않고 `해당없음`으로 처리하며, 실제 `납부`된 금액만 수납 실적으로 보존합니다.
